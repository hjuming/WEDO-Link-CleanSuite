# CleanSuite v1.2

CleanSuite is a comprehensive macOS system cleaning utility built with SwiftUI.

## Features
- **Dashboard**: Visual disk usage monitoring.
- **Quick Clean**: Fast system cleanup.
- **Deep Clean**: Thorough system maintenance.
- **Mail & Browser Clean**: Specialized cleaning for Mail and Browsers.
- **LINE Clean**: Cleanup for LINE application data.
- **Scheduler**: Automated cleaning schedules.

## Structure
- `CleanSuiteApp/`: Main SwiftUI application.
- `CleanSuiteHelper/`: Helper tool for privileged operations.
- `CLI/`: Command-line interface tools.
- `pkg/`: Installer package resources.

## Building
Open the `CleanSuite` folder in Xcode. This project is configured as a Swift Package for easy compatibility.
