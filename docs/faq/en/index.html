Frequently Asked Questions

Answers to common questions about CleanSuite

1. What is CleanSuite?

CleanSuite is a lightweight system cleaning utility designed for macOS. It provides Quick Clean, Deep Clean, and privacy cleaning features to free up disk space and keep your system running smoothly.

2. How do I download and update it?

You can download the latest installer directly from our homepage or GitHub Releases. CleanSuite includes an automatic update checker that will notify you when a new version is available.

3. Why does macOS block the app?

Because CleanSuite is not yet notarized by Apple. This is a standard macOS security feature.

To bypass this: Right-click (or Control-click) on CleanSuite.app in your Applications folder, select "Open", and then click "Open" in the dialog box.

4. Is the unsigned version safe?

Yes, CleanSuite is open-source software. All code is transparently available on GitHub, so you can verify that there is no malicious behavior.

5. How can I submit feedback or bug reports?

Please visit our GitHub Issues page to report bugs or suggest new features.
