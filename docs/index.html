專為 macOS 設計的輕量級系統清理工具。

支援 macOS 13.0 (Ventura) 及以上版本

強大功能，簡單易用

⚡️

快速清理

一鍵掃描並移除系統暫存檔、快取與日誌，瞬間釋放磁碟空間。

🧹

深度清潔

深入系統底層，找出被遺忘的大型垃圾檔案與陳舊資料。

🛡

隱私保護

清除瀏覽器 (Safari, Chrome) 與通訊軟體 (LINE) 的快取與紀錄。

自動排程

設定每日、每週或每月自動清理，讓系統維護全自動化。

郵件清理

專門針對 macOS 郵件 App 的附件與快取進行瘦身。

個人化設定

自訂清理規則與通知偏好，打造最適合你的清理工具。
