3. Setting up Auto Scheduler

Don't want to clean manually? Set up an automatic schedule to let CleanSuite work in the background.

  1. Open CleanSuite and click Scheduler in the sidebar.
  2. Choose frequency: Daily, Weekly, or Monthly.
  3. Toggle the switch on. The system will automatically run Quick Clean tasks.
