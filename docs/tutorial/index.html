CleanSuite 使用教學

快速上手指南：從安裝到進階設定

1. 下載與安裝

從首頁或 GitHub Releases 下載最新的 CleanSuite-Installer.pkg 檔案。

💡 提示：無法開啟應用程式？

如果 macOS 顯示「無法打開，因為它來自未識別的開發者」，請前往 系統設定 > 隱私權與安全性，往下滑動並點選 「強制打開」 即可。

2. 功能模組介紹

⚡️ 快速清理 (Quick Clean)

適合日常使用。快速掃描並刪除系統快取、暫存檔與日誌檔。建議每週執行一次。

🧹 深度清潔 (Deep Clean)

進行更全面的掃描，包含 Xcode 垃圾、舊的 iOS 更新檔等大型垃圾。建議每月執行一次。

🌐 瀏覽器清理

支援 Safari, Chrome, Edge。清除瀏覽紀錄、Cookies 與快取圖片，保護隱私並釋放空間。

📨 郵件與通訊軟體

清除 Mail App 下載的附件檔，以及 LINE 的快取資料（不會刪除聊天記錄）。

3. 設定自動排程

不想手動清理？你可以設定自動排程，讓 CleanSuite 在背景默默為你工作。

  1. 開啟 CleanSuite，點選側邊欄的 排程清理。
  2. 選擇頻率：每日、每週 或 每月。
  3. 開啟開關，系統將會自動在背景執行快速清理任務。
