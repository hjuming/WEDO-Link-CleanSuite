CleanSuite 使用教學

一步一步帶你完成系統清理、管理與最佳化

1. 安裝 CleanSuite

下載網站提供的 .pkg 安裝檔，雙擊後依照畫面指示完成安裝即可。 安裝完成後，你可以在「應用程式」資料夾找到 CleanSuite。

2. 主要介面說明

  • 快速清理：清除快取、暫存檔案、系統垃圾。
  • 深度清潔：更全面的系統清理，適合長期未清理者。
  • App 清理：快速清除 LINE、Mail、Safari/Chrome 等資料。
  • 排程清理：自動每日、每周或每月執行清理。
  • 設定：切換偏好設定、清理等級、通知選項。

3. 清理模式介紹

快速清理：建議每日或工作天使用。

深度清潔：建議每週或每月一次（視使用量）。

App 清理：當 LINE 或 Mail 空間異常膨脹時使用。

4. 排程清理設定

開啟 CleanSuite → 點選 排程清理 → 選擇執行頻率（每日/每週/每月）。 macOS 將自動在背景執行清理任務，無須手動操作。

5. 常見問題

  • Q: 清理後空間沒有明顯變大？
    可能是你的硬碟使用情況已經很乾淨，建議使用「深度清潔」。
  • Q: LINE 或 Mail 清不乾淨？
    請先關閉該 App，再重新執行清理。
  • Q: 排程沒有生效？
    請確認 macOS 背景權限與電腦未完全關機。
